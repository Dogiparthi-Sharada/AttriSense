## What got refreshed vs. v1

- Launch post and Twitter thread now lead with **"five modalities,"**
  not three. Cox PH and EconML T-Learner each get their own beat.
- Launch post highlights the **Review_ID identification-bias
  mitigation** — this is the single most differentiating feature
  on a student-built workforce-analytics project, and v1 missed it.
- Engagement bank adds canned replies for the inevitable "but isn't
  this just attrition prediction?" comment — the answer is
  *causal uplift + survival + fairness gate*, all of which v1
  didn't have.
- Pipeline tracker now tracks **5 outcomes**: recruiter intro,
  hiring-manager call, technical screen, on-site, offer — instead
  of v1's binary "responded / didn't."
