# Pipeline Tracker — populate during Week 2, review every Saturday

> **Format:** plain markdown table — survives `mkdocs --strict`,
> grep-able, version-controllable, no spreadsheet lock-in.
>
> **Discipline:** every conversation, every DM, every "no thanks"
> goes in. Especially the no's — they tell you which framing isn't
> landing.

---

## Stages

| Stage | Definition |
|-------|------------|
| **Inbound** | They messaged you first |
| **Outbound** | You messaged them first |
| **Replied** | They responded ≥ once |
| **Call booked** | Calendar invite exists for a 15+ min conversation |
| **Call done** | Conversation happened, no follow-up scheduled yet |
| **Next step** | Concrete action exists — referral, intro, next round, take-home |
| **Offer** | Written offer in hand |
| **Closed** | Either accepted, declined, or fully ghosted (>10 business days) |
